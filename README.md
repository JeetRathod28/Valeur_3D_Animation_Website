# VALEUR 3D Bag Website

> **A luxury brand experience, engineered for the web.**
>
> _Next.js 16 • React 19 • TypeScript • Tailwind CSS • Framer Motion • 3D Animation_

---

## ✨ Project Overview

VALEUR is a luxury leather goods house. This site is a visual storytelling platform that blends:

- **3D animation** (40-frame hero sequence, scroll-driven)
- **Parallax & scroll effects** for immersive product reveals
- **Minimalist, neo-brutalist design** with custom CSS variables
- **Performance-first**: Next.js App Router, image optimization, and server components

---

## 🚀 Features

- **Hero 3D Animation:** 40-frame sequence, buttery-smooth, scroll-synced
- **Sectioned Architecture:** Each homepage section is a modular React component
- **Framer Motion:** Advanced scroll, parallax, and reveal animations
- **Tailwind + Custom CSS:** Design system with CSS variables for palette/typography
- **Next.js Image Optimization:** All images managed via `<Image />`
- **TypeScript:** End-to-end type safety
- **Modern Linting:** Core Web Vitals, strict TypeScript, and Next.js rules

---

## 🏗️ Project Structure

- `src/app/` — App router, layout, error, not-found, and main page
- `src/components/sections/` — Homepage sections (HeroSplit, ValueProp, MechanismReveal, MaterialDetail, DayNightTransition, Philosophy)
- `public/frames/` — 3D animation frames for Hero section
- `public/images/` — Product and detail images
- `globals.css` — Custom design system, Tailwind, and utility classes

---

## 🛠️ Getting Started

```bash
# 1. Install dependencies
npm install

# 2. Run the development server
npm run dev
# or: yarn dev / pnpm dev / bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
